# flet-supertab
SuperTab control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "flet-supertab @ git+https://github.com/MyGithubAccount/flet-supertab",
  "flet>=0.80.2",
]
```

* **uv/pip dependency**  

If the package is published on pypi.org:

```
dependencies = [
  "flet-supertab",
  "flet>=0.80.2",
]
```

Build your app:
```
flet build macos -v
```
